# EMI Calculator:
> It's a EMI calculator which is made using HTML,CSS,JS . Here are 3 options -->
- Loan Amount
- interest rate/year
- Duration
These 3 will be taken from users.

- Monthly Installment 
- Monthly Interest
- Total Repayment 
- Total Interest Cost
